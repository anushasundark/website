Multitask Deeplearning for Predicting poverty using satellite images

In this Main project, we developed a "predicting poverty using satellite images"

This project addresses the challenges posed by the lack of reliable data on global economic livelihood, especially in developing areas, by leveraging overhead satellite imagery for poverty level estimation.

Key Components:

  • Deep learning computer vision methods
  • Prediction of poverty levels from overhead satellite images
  • Experimentation with both daytime and nighttime imagery
  • Exploration of data quantity and augmentation impact on network performance
  • Evaluation of network robustness on data from continents not present in the development set
